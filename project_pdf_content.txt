Objective
Develop a native mobile application (Android & iOS) that embeds a Flutter module, passing
runtime configurations such as theming, UUID/session , and typography. The Flutter
module should display a list and allow navigation to detail screens, support an external
payment flow and return payloads back to the native side.
Here’s a clean, formatted version of your assignment for better clarity and execution. This
can be shared as a task document or a project brief with a team:

Navigation Flow
1. Launch App
a. If UUID is already generated & cached, bypass login and directly load the
Flutter module
b. Else, show the Login Screen
2. Login Screen
a. Simple form with basic validation
b. On success:
i. Generate a random UUID
ii. Pass app configurations + UUID to the Flutter module
iii. Load the Flutter module
